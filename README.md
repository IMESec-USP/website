# IMEsec Website

The website for the infosec group IMEsec. [https://imesec.ime.usp.br](https://imesec.ime.usp.br).

### To develop the front-end

After cloning the repository, run
```
npm i
bundle install
```

To develop, run
```
npx webpack -w
```
In one console, and
```
bundle exec jekyll serve
```
in another.
